✅ Implemented solution

Solution implemented
with Sciber

Three combined measures neutralised the attack and restored accessibility.

🔀

Setup an Application Gateway

Azure Application Gateway configured as central traffic management. Intelligently distributes requests and absorbs volumetric attack spikes.

🛡

Setup Web Application Firewall (WAF)

OWASP ruleset activated. Detects and blocks bot traffic, scrapers and known attack patterns at application level.

🌎

Block all traffic from China

Geo-blocking at gateway level. All connections from China blocked — immediate CPU relief and end of the attack.
